# 快速開始指南

## 啟動應用

```bash
npm run dev
```

應用將在 `http://localhost:3000` 啟動（如果端口被占用會自動嘗試下一個端口）。

## 測試新的篩選功能

### 步驟 1: 加載測試數據

1. 打開瀏覽器訪問應用
2. 點擊「文件上傳」或「JSON 輸入」
3. 加載提供的測試數據文件：`/public/sample-data-employees.json`

或直接貼上以下 JSON：

```json
[
  {
    "id": 1,
    "name": "張三",
    "age": 25,
    "salary": 50000,
    "department": "技術部",
    "active": true,
    "city": "台北"
  },
  {
    "id": 2,
    "name": "李四",
    "age": 30,
    "salary": 60000,
    "department": "業務部",
    "active": true,
    "city": "台中"
  },
  {
    "id": 3,
    "name": "王五",
    "age": 28,
    "salary": 55000,
    "department": "技術部",
    "active": false,
    "city": "台北"
  }
]
```

### 步驟 2: 切換到表格視圖

1. 在「數據預覽」區域，點擊「表格視圖」按鈕
2. 你會看到數據以表格形式展示

### 步驟 3: 開啟篩選面板

1. 點擊「篩選」按鈕（帶有篩選圖示）
2. 篩選面板會展開，顯示所有可用的字段

### 步驟 4: 測試不同類型的篩選

#### 測試數字範圍篩選（年齡）
1. 找到 `age` 字段
2. 在最小值輸入 `25`
3. 在最大值輸入 `29`
4. 觀察表格只顯示年齡在 25-29 之間的記錄

#### 測試文字搜索（部門）
1. 找到 `department` 字段
2. 輸入「技術部」
3. 觀察表格只顯示技術部的員工

#### 測試布林篩選（活躍狀態）
1. 找到 `active` 字段
2. 選擇「是 (true)」
3. 觀察只顯示活躍的員工

### 步驟 5: 組合多個篩選

1. 同時設置：
   - `department`: 技術部
   - `active`: 是
   - `salary`: 最小值 50000
2. 觀察符合所有條件的員工

### 步驟 6: 測試分組功能

1. 在「分組依據」下拉選單選擇 `city`
2. 觀察數據按城市分組
3. 點擊分組標題可展開/收起
4. 注意分組會考慮當前的篩選條件

### 步驟 7: 管理篩選

1. 觀察篩選面板上方的徽章顯示活躍的篩選
2. 點擊徽章上的 X 按鈕可移除單一篩選
3. 點擊「清除所有篩選」按鈕重置所有篩選

## 常見問題

### Q: 篩選後看不到數據？
A: 檢查篩選條件是否過於嚴格，嘗試放寬條件或清除所有篩選。

### Q: 如何查看所有唯一值？
A: 開啟篩選面板，每個字段下方會顯示「X 個唯一值」。

### Q: 分組後能否再篩選？
A: 可以，篩選和分組可以組合使用，分組會基於篩選後的數據。

### Q: 如何找到高薪員工？
A: 在 `salary` 字段設置最小值，例如 60000。

### Q: 支持多級分組嗎？
A: 目前僅支持單一字段分組，多級分組可作為未來增強功能。

## 性能建議

- 對於大量數據（>1000 行），建議先設置篩選條件再進行分組
- 使用分頁功能瀏覽大量結果
- 隱藏不需要的字段可提升表格渲染性能
