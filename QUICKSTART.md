# 快速啟動

## 三步開始使用

### 1. 安裝依賴（僅首次需要）

```bash
npm install
```

### 2. 啟動開發服務器

```bash
npm run dev
```

### 3. 訪問應用

打開瀏覽器訪問：**http://localhost:3000**

---

## 快速測試

1. 點擊「直接輸入」標籤
2. 複製以下 JSON 並粘貼到文本框：

```json
{
  "name": "測試數據",
  "items": ["項目1", "項目2", "項目3"],
  "count": 3
}
```

3. 點擊「解析 JSON」
4. 查看預覽
5. 點擊「確認保存數據」

完成！現在可以在右側看到歷史記錄。

---

## 文件上傳測試

項目包含範例數據文件：`public/sample-data.json`

1. 點擊「文件上傳」標籤
2. 點擊「選擇文件」
3. 選擇 `public/sample-data.json`
4. 查看自動解析的結果
